## üìñ Overview

**MAEPLE** is not just another mood tracker. It is a **Professional-Grade Health Intelligence Engine** designed to bridge the gap between subjective experience and objective biological signals.

Built with a **neuro-affirming philosophy**, MAEPLE helps users‚Äîespecially those with ADHD, Autism, or CPTSD‚Äîmove beyond "symptom surveillance" and towards "pattern literacy." We don't ask "What's wrong with you?"; we ask "What is your context?"

### Why MAEPLE?

-   **üîç Objective Reality Checks**: Our **Bio-Mirror** technology uses AI vision to detect signs of stress (jaw tension, eye fatigue) that you might be masking, even from yourself.
-   **üîã Capacity > Mood**: We track a 7-point **Capacity Grid** (Focus, Social, Sensory, etc.) using metrics like **Bandwidth**, **Load**, and **Interference**.
-   **üîí Privacy First**: Your biometric data is yours. Analysis is performed with strict privacy controls.

## üöÄ Quick Start

Get MAEPLE running in minutes using Docker.

### Prerequisites
-   Docker & Docker Compose
-   Git

### Installation

1.  **Clone the repo**
    ```bash
    git clone https://github.com/your-org/maeple.git
    cd maeple
    ```

2.  **Setup Environment**
    ```bash
    cp .env.example .env
    # Edit .env with your API keys
    ```

3.  **Run with Docker**
    ```bash
    cd deploy
    docker compose up -d --build
    ```

4.  **Access the App**
    -   Frontend: `http://localhost`
    -   API Health: `http://localhost/api/health`
